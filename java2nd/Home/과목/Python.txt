1. 문자열 포매팅 
-> 이름 : 가변 매개변수, 인덱스 : 일반 매개변수
-> 매개변수는 가변, 일반 순이 되지만 인수값은 가변,일반 순으로가 불가능

2. 리스트
-> 2차, 3차리스트를 다른 리스트에 옮겨담을때 주소값만 옮길수있는데, 해당 값만 옮기려고할때 방법은?
-> ex) list1 = [] + list2 해당 형식으로하면 주소값이 아닌 새로운값 생성 후 넣기때문에 같이 변동이 안됨
-> 얕은 복사(shallow copy)와 깊은 복사(deep copy)의 차이

포맷코드 -> 정렬 공백시 문자열, 숫자, 실수 전부 우측 정렬
포매팅 -> 정렬 공백시 숫자,실수 우측정렬   , 문자열은 좌측정렬


함수 이름 - applePie -> apply_pie
			ex) 자바처럼 대문자 중간에 안넣음.. all 소문자

매개변수로 *매개변수명 쓰면 여러개를 전달가능 리스트형태가 아님
ex)해당 함수에서 for문을 쓸 경우 활용

결과 값은 언제나 하나
-> return으로 ,으로 구분하여 2개 이상의 값을 반환했을시 리스트형태의 튜플값으로 반환
-> 즉 , 결과 값은 1개

lambda 예약어 - 함수를 한줄로 간결하게 만들때 사용, return이 없어도 결과값을 돌려줌
					  - 함수의 이름이 없음, 일회용
ex ) lambda a, b : a+b


클래스 - 무엇인가를 만들려고 할때의 도면
		   - 변수, 메서드들의 집합체
객체 - 해당 클래스로 만든 변수,함수 집합?
인스턴스 - 해당 객체는 클래스의 인스턴스
메서드 - 클래스안의 함수들을 메서드라고 한다.


생성자 - 클래스가 생성될때 자동으로 실행되는 함수
상속 - 자식클래스가 부모클래스의 것을 그대로 가져오는것
오버라이딩 - 자식클래스가 부모클래스로부터 가져온것을 수정하는것
객체변수 - 객체를 생성할때 그 각자의 객체들이 가진 객체안의 변수
클래스변수 - 객체를 여러개 생성하더라도 같은 클래스의 변수일 경우, 해당 변수는 공유

모듈 - 함수나 변수or 클래스를 모아놓은 파일 

||, && 연산자 없음.

** 오버로딩 - 같은 이름의 메서드를 매개변수 갯수만 다르게 하여 여러번 정의할 수 있는 것
